## 🏗️ 框架架构

1. **`optimized_integration_engine.py`** (60KB)

## 🎯 技术实现特点

### 关键技术决策

1. **置信度阈值**：0.6确保高质量信号，降低虚假交易风险
2. **磁力位合并**：2%价格容忍度合并相近关键水平
3. **验证权重**：触及次数25%，成交量20%，价格行为25%，时间框架15%，波动率15%
4. **风险参数**：单笔交易最大风险2%，最小风险回报比1.5

### 性能特点

1. **实时分析**：100个数据点分析仅需<1秒
2. **结果持久化**：JSON格式保存完整分析结果和交易计划
3. **模块化架构**：6个独立模块，易于维护和扩展
4. **可移植性**：仅依赖numpy+pandas，可在任何Python环境运行

## 📝 使用指南

### 快速开始

```python
# 1. 导入框架
from optimized_integration_engine import OptimizedPriceActionIntegrationEngine
from price_action_rules_integrator import PriceActionRulesIntegrator

# 2. 加载数据
engine = OptimizedPriceActionIntegrationEngine()

# 3. 运行分析
analysis_results = engine.run_analysis()

# 4. 整合价格行为规则
integrator = PriceActionRulesIntegrator()
enhanced_results = integrator.integrate_rules(analysis_results, your_price_data)

# 5. 生成报告
integration_report = integrator.generate_integration_report(enhanced_results)
```
