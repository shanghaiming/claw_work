## 趋势篇第23章：开盘起趋势和小幅回撤趋势
### 核心概念
1. **开盘起趋势日定义**：
   - 多头趋势日的低点和空头趋势日的高点在当天头几棒内形成
   - 如果当日开盘区间小于近日平均日区间的25%，可能是多头趋势日的双重底或空头趋势日的双重顶
   - 如果开盘区间约为平均日区间的50%，更可能出现突破，引出趋势型交易区间日

2. **关键特征**：
   - 大型缺口开盘常引起从开盘起的趋势日（上涨缺口：60%多头趋势日，40%空头趋势日）
   - 开盘起趋势日具有紧迫感，通常是最强的趋势，回撤幅度最小
   - 第一棒成为当日高点/低点的几率：大型开盘缺口时约50%（如果第一棒是强趋势棒）

3. **小幅回撤趋势日**：
   - 最强类型的开盘起趋势日：开盘区间很小，趋势持续整天，只有很小回撤
   - 回撤可能只有10-12个跳动（平均日区间的10%-30%）
   - 最后一两小时内通常出现较大回撤（早期回撤的150%-200%），然后趋势恢复

4. **交易挑战**：
   - 很多反向趋势棒，初学者易寻找反转而非顺势交易
   - 趋势常处于紧凑通道内，回撤常击中盈亏平衡止损
   - 波段架构成功率可达70%+，但信号棒常看起来很糟糕

### 可量化规则
```python
# 开盘起趋势日识别
def identify_opening_drive_day(df, daily_avg_range, first_hour_bars=12):
    """
    识别开盘起趋势日
    返回：{'is_opening_drive': bool, 'direction': 'bullish'/'bearish', 'confidence': 0-1}
    """
    if len(df) < first_hour_bars:
        return None
    
    # 计算开盘区间（前几棒）
    opening_bars = min(first_hour_bars, len(df))
    opening_high = df['high'].iloc[:opening_bars].max()
    opening_low = df['low'].iloc[:opening_bars].min()
    opening_range = opening_high - opening_low
    
    # 开盘区间与日平均区间比例
    range_ratio = opening_range / daily_avg_range
    
    # 检查是否为开盘起趋势日
    if range_ratio < 0.25:  # 小于25%
        # 检查第一棒方向
        first_bar = df.iloc[0]
        is_bullish_trend_bar = first_bar['close'] > first_bar['open'] and \
                              abs(first_bar['close'] - first_bar['open']) / (first_bar['high'] - first_bar['low']) > 0.6
        is_bearish_trend_bar = first_bar['close'] < first_bar['open'] and \
                              abs(first_bar['close'] - first_bar['open']) / (first_bar['high'] - first_bar['low']) > 0.6
        
        if is_bullish_trend_bar:
            return {'is_opening_drive': True, 'direction': 'bullish', 'confidence': 0.7}
        elif is_bearish_trend_bar:
            return {'is_opening_drive': True, 'direction': 'bearish', 'confidence': 0.7}
    
    return {'is_opening_drive': False, 'confidence': 0.3}
```

## 趋势篇第26章：台阶形态 - 宽通道趋势
### 核心概念
1. **台阶日定义**：
   - 趋势型交易区间日的变种，至少包含三段交易区间
   - 当天出现很宽的波段，但波段高点和低点呈现趋势变化
   - 双向交易活跃，但一方有更多控制权

2. **关键特征**：
   - 每次突破后都形成超越突破点的回撤（突破测试）
   - 相邻波段之间存在重叠
   - 如果每个突破都比前一个小，形成逐渐收缩的台阶形态，表明动能衰退

3. **交易策略**：
   - 可在两个方向下单，但尽量波段化顺势交易
   - 经验丰富者可在趋势通道线附近买进，趋势线附近卖空
   - 使用限价单在特定跳动数处入场（如前一波段的3-4点下方买进）

### 可量化规则
```python
# 台阶形态识别
def identify_stairs_pattern(df, lookback=40):
    """
    识别台阶形态（宽通道趋势）
    返回：{'is_stairs': bool, 'direction': 'bullish'/'bearish', 'channel_width': float, 'is_contracting': bool}
    """
    if len(df) < lookback:
        return None
    
    # 寻找波段高点和低点
    highs = find_peak_pivots(df['high'], min(10, lookback))
    lows = find_valley_pivots(df['low'], min(10, lookback))
    
    if len(highs) < 3 or len(lows) < 3:
        return None
    
    # 检查是否为台阶形态（依次更高的高点/低点或更低的高点/低点）
    is_bullish_stairs = True
    is_bearish_stairs = True
    
    # 检查多头台阶（更高高点和更高低点）
    for i in range(1, min(3, len(highs))):
        if df.iloc[highs[i]]['high'] <= df.iloc[highs[i-1]]['high']:
            is_bullish_stairs = False
            break
    
    for i in range(1, min(3, len(lows))):
        if df.iloc[lows[i]]['low'] <= df.iloc[lows[i-1]]['low']:
            is_bullish_stairs = False
            break
    
    # 检查空头台阶（更低高点和更低低点）
    for i in range(1, min(3, len(highs))):
        if df.iloc[highs[i]]['high'] >= df.iloc[highs[i-1]]['high']:
            is_bearish_stairs = False
            break
    
    for i in range(1, min(3, len(lows))):
        if df.iloc[lows[i]]['low'] >= df.iloc[lows[i-1]]['low']:
            is_bearish_stairs = False
            break
    
    if is_bullish_stairs:
        # 计算通道宽度
        channel_width = (df.iloc[highs[-1]]['high'] - df.iloc[lows[-1]]['low']) / df.iloc[lows[-1]]['low']
        
        # 检查是否为收缩台阶
        is_contracting = False
        if len(highs) >= 3:
            move1 = df.iloc[highs[1]]['high'] - df.iloc[highs[0]]['high']
            move2 = df.iloc[highs[2]]['high'] - df.iloc[highs[1]]['high']
            if move2 < move1 * 0.8:  # 第二波上涨小于第一波的80%
                is_contracting = True
        
        return {'is_stairs': True, 'direction': 'bullish', 'channel_width': channel_width, 'is_contracting': is_contracting}
    
    elif is_bearish_stairs:
        channel_width = (df.iloc[highs[-1]]['high'] - df.iloc[lows[-1]]['low']) / df.iloc[highs[-1]]['high']
        
        is_contracting = False
        if len(lows) >= 3:
            move1 = df.iloc[lows[0]]['low'] - df.iloc[lows[1]]['low']
            move2 = df.iloc[lows[1]]['low'] - df.iloc[lows[2]]['low']
            if move2 < move1 * 0.8:
                is_contracting = True
        
        return {'is_stairs': True, 'direction': 'bearish', 'channel_width': channel_width, 'is_contracting': is_contracting}
    
    return {'is_stairs': False}
```
