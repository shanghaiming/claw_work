## AL Brooks价格行为交易核心方法论总结

### 🎯 核心哲学
1. **纯价格行为**：不依赖技术指标，只看K线（棒线）、成交量和市场结构
2. **市场逻辑**：所有价格运动都反映了多空力量的实时博弈
3. **概率思维**：寻找高概率交易设置，接受亏损是交易的一部分
4. **上下文分析**：同样的K线形态在不同市场环境中意义不同

### 📊 价格行为频谱理论
- **极端趋势**：几乎每个跳动都比前一个跳动高（多头）或低（空头）
- **极端交易区间**：每一两个向上跳动都跟随一两个向下跳动（双向市场）
- 大多数时间市场处于两者之间的状态

### 🔄 市场惯性原则
- **趋势倾向于继续**：如果市场处于趋势中，大多数反转尝试都将失败
- **区间倾向于维持**：如果市场处于交易区间，大多数突破尝试都将失败
- **80%法则**：趋势中的反转尝试有80%会失败，变为顺势旗形

### 🏗️ 关键市场结构概念
1. **总在场内（Always In）**：始终知道市场方向倾向，主要在趋势方向上交易
2. **腿的计数系统**：使用高点1、高点2、低点1、低点2等标记市场结构
3. **尖峰和通道**：趋势通常由急剧的尖峰开始，然后转为较缓的通道
4. **测试概念**：市场返回关键支撑/阻力位（趋势线、波段点、先前高低点）
5. **高潮识别**：趋势耗尽信号，通常表现为连续的趋势棒后出现暂停棒

### 📈 关键交易理念
1. **测试的重要性**：市场两次尝试做某事都失败后，通常会尝试相反方向
2. **突破vs反转的区分**：可靠区分成功突破和失败突破（反转）
3. **信号棒质量**：最好的信号棒是交易方向上的强势趋势棒
4. **顺势交易优先**：初学者应该只在趋势方向上交易，使用趋势方向上的趋势棒作为信号棒

### 🔢 可量化的交易规则
- **市场状态识别**：基于趋势棒比例、十字星比例、棒线重叠程度判断趋势/区间
- **反转棒入场规则**：基于实体大小、尾线长度、与前棒重叠程度、收盘价位置综合评分
- **突破入场规则**：在趋势方向上突破关键价位（前期高点/低点）
- **风险管理规则**：基于信号类型和市场状态动态设置止损
- **仓位管理规则**：基于趋势强度和信号质量动态调整仓位大小
- **出场规则**：基于市场状态设置不同风险回报比的目标位

### 第6章：信号棒 - 其他类型
#### 核心概念
1. **强趋势棒作为信号棒**：在趋势尖峰阶段，强趋势棒本身可作为信号棒。交易者可在趋势棒收盘时或上方1个跳动处入场。
2. **反转形态的多样性**：除经典反转棒外，还有多种其他反转架构：
   - **双棒反转**：最常见的反转架构之一，由两条相邻的趋势棒组成（方向相反）。所有反转本质上都是双棒反转。
   - **三棒反转**：双棒反转的变种，中间包含一条普通棒（窄幅棒或十字星）。
3. **窄幅棒作为信号棒**：
   - **内包棒**：波幅较窄，完全位于前一棒范围内。
   - **ii/iii形态**：连续两条（ii）或三条（iii）内包棒。
   - **ioi形态**：内包-外包-内包形态。
4. **信号棒质量原则**：
   - 顺势交易对信号棒要求较低，逆势交易对信号棒要求较高。
   - 最好的信号棒是交易方向上的趋势棒。
   - 趋势越强，顺势交易对强信号棒的要求越低。

#### 可量化规则
```python
# 双棒反转识别
def identify_two_bar_reversal(df, idx):
    """
    识别双棒反转
    规则：两条相邻棒线，方向相反，都是趋势棒
    """
    bar1 = df.iloc[idx-1]
    bar2 = df.iloc[idx]
    
    # 检查趋势棒特征
    is_trend_bar1 = abs(bar1['close'] - bar1['open']) / (bar1['high'] - bar1['low']) > 0.6
    is_trend_bar2 = abs(bar2['close'] - bar2['open']) / (bar2['high'] - bar2['low']) > 0.6
    
    # 检查方向相反
    direction1 = 'bullish' if bar1['close'] > bar1['open'] else 'bearish'
    direction2 = 'bullish' if bar2['close'] > bar2['open'] else 'bearish'
    
    if is_trend_bar1 and is_trend_bar2 and direction1 != direction2:
        return f'{direction2}_two_bar_reversal'
    return None

# ii/iii形态识别
def identify_ii_pattern(df, idx):
    """
    识别ii（连续内包棒）形态
    """
    if idx < 2:
        return None
    
    bar1 = df.iloc[idx-2]
    bar2 = df.iloc[idx-1]
    bar3 = df.iloc[idx]
    
    # 检查内包关系
    is_inside1 = (bar2['high'] <= bar1['high'] and bar2['low'] >= bar1['low'])
    is_inside2 = (bar3['high'] <= bar2['high'] and bar3['low'] >= bar2['low'])
    
    if is_inside1 and is_inside2:
        # 检查窄幅特征
        range_ratio1 = (bar2['high'] - bar2['low']) / (bar1['high'] - bar1['low'])
        range_ratio2 = (bar3['high'] - bar3['low']) / (bar2['high'] - bar2['low'])
        
        if range_ratio1 < 0.8 and range_ratio2 < 0.8:
            return 'ii_pattern'
    
    return None
```

### 第7章：外包棒
#### 核心概念
1. **外包棒定义**：高点高于前一棒高点，低点低于前一棒低点。
2. **外包棒本质**：单棒交易区间，需要结合背景分析。
3. **外包棒的风险**：
   - 常作为陷阱出现，特别是在交易区间中。
   - 突破外包棒入场风险较高，因为止损距离较大。
4. **ioi形态**：内包-外包-内包形态，可作为突破信号。
5. **外包棒的合理使用场景**：
   - 当外包棒前一棒是好的信号棒时。
   - 在强反转中作为二次入场棒。
   - 在趋势反转的第一条腿中，外包棒作用类似强趋势棒。

#### 可量化规则
```python
# 外包棒识别与风险评估
def assess_outside_bar_risk(df, idx, market_state):
    """
    评估外包棒交易风险
    返回：risk_score (0-1, 越高风险越大)
    """
    bar = df.iloc[idx]
    prev_bar = df.iloc[idx-1]
    
    # 检查是否为外包棒
    is_outside = (bar['high'] > prev_bar['high'] and bar['low'] < prev_bar['low'])
    
    if not is_outside:
        return 0
    
    # 风险评估因素
    factors = []
    
    # 1. 实体大小（实体越小，风险越高）
    body_ratio = abs(bar['close'] - bar['open']) / (bar['high'] - bar['low'])
    factors.append(1.0 - body_ratio)  # 实体越小，风险越高
    
    # 2. 与前一棒重叠程度（重叠越多，风险越高）
    overlap = calculate_overlap(bar, prev_bar)
    factors.append(overlap)
    
    # 3. 市场状态（交易区间中风险更高）
    if market_state == 'trading_range':
        factors.append(0.8)
    elif market_state == 'strong_trend':
        factors.append(0.2)
    else:
        factors.append(0.5)
    
    # 4. 位置（在区间中部风险更高）
    position_score = calculate_position_score(df, idx)
    factors.append(position_score)
    
    # 综合风险评分
    risk_score = sum(factors) / len(factors)
    return risk_score

# ioi形态识别
def identify_ioi_pattern(df, idx):
    """
    识别ioi（内包-外包-内包）形态
    """
    if idx < 2:
        return None
    
    bar1 = df.iloc[idx-2]  # 第一根内包棒
    bar2 = df.iloc[idx-1]  # 外包棒
    bar3 = df.iloc[idx]    # 第二根内包棒
    
    # 检查内包关系
    is_inside1 = (bar1['high'] <= bar2['high'] and bar1['low'] >= bar2['low'])
    is_inside2 = (bar3['high'] <= bar2['high'] and bar3['low'] >= bar2['low'])
    
    # 检查外包关系
    is_outside = (bar2['high'] > bar1['high'] and bar2['low'] < bar1['low'])
    
    if is_inside1 and is_outside and is_inside2:
        return 'ioi_pattern'
    
    return None
```

### 第8章：棒线收盘价的重要性
#### 核心概念
1. **等待收盘确认**：避免在棒线收盘前过早入场，等待收盘价确认信号。
2. **收盘前的风险**：
   - 棒线在收盘前几秒可能反转，导致过早入场亏损。
   - 强趋势中可适当提早入场，但需谨慎。
3. **收盘价的意义**：
   - 机构交易者关注收盘价（线图使用收盘价）。
   - 强收盘表明趋势强度，弱收盘表明犹豫。
4. **时间框架选择**：
   - 较小时间框架（如3分钟图）可能导致更多止损被触发。
   - 坚持使用单一时间框架，避免矛盾信号。

#### 可量化规则
```python
# 收盘价强度评估
def assess_close_strength(df, idx, trend_direction):
    """
    评估收盘价强度
    返回：close_strength (0-1, 越高表示收盘越强)
    """
    bar = df.iloc[idx]
    
    if trend_direction == 'bullish':
        # 多头趋势中：收盘价越高越强
        close_strength = (bar['close'] - bar['low']) / (bar['high'] - bar['low'])
    elif trend_direction == 'bearish':
        # 空头趋势中：收盘价越低越强
        close_strength = (bar['high'] - bar['close']) / (bar['high'] - bar['low'])
    else:
        # 无明确趋势：收盘价靠近极端值表示强度
        close_to_high = (bar['close'] - bar['low']) / (bar['high'] - bar['low'])
        close_to_low = (bar['high'] - bar['close']) / (bar['high'] - bar['low'])
        close_strength = max(close_to_high, close_to_low)
    
    return close_strength

# 入场时机决策
def determine_entry_timing(df, idx, signal_type, market_volatility):
    """
    决定入场时机（立即入场 vs 等待收盘）
    返回：'immediate', 'wait_for_close', 或 'avoid'
    """
    bar = df.iloc[idx]
    
    # 基础规则
    if market_volatility > 0.8:
        # 高波动性市场：等待收盘
        return 'wait_for_close'
    
    if signal_type in ['reversal', 'breakout']:
        # 反转和突破信号：通常等待收盘确认
        return 'wait_for_close'
    
    if signal_type == 'trend_continuation':
        # 趋势延续：在强趋势中可考虑立即入场
        current_trend = assess_trend_strength(df, idx)
        if current_trend > 0.7:
            return 'immediate'
        else:
            return 'wait_for_close'
    
    # 默认等待收盘
    return 'wait_for_close'
```

### 第10章：二次入场
#### 核心概念
1. **二次入场原则**：
   - 二次入场通常比首次入场更可靠。
   - "好的执行等价于不好的交易"：如果二次入场价位比首次入场更好，可能是陷阱。
2. **强趋势中的二次入场**：
   - 在强趋势中，应等待二次反转信号再考虑逆势交易。
   - 首次反转尝试往往失败。
3. **入场价位管理**：
   - 二次入场价位通常与首次入场相同或更差。
   - 更好的价位可能意味着市场在欺骗。

#### 可量化规则

### 第13章：趋势线
#### 核心概念
1. **趋势线定义**：
   - 多头趋势线：经过多头趋势低点的直线
   - 空头趋势线：经过空头趋势高点的直线
2. **绘制方法**：
   - 波段点绘制
   - 最佳拟合技术（线性回归）
   - 最接近的直线
   - 趋势通道线平移
3. **趋势线的作用**：
   - 趋势方向上的回撤入场
   - 趋势线突破后的逆势入场
   - 趋势动能变化的指示器（斜率逐渐变平）
4. **趋势线突破的意义**：
   - 市场不再由单方控制的第一个信号
   - 双向交易几率大大增加
   - 大多数突破失败

#### 可量化规则
```python
# 趋势线绘制算法
def draw_trend_line(df, pivot_type='low', lookback=20, min_pivots=2):
    """
    绘制趋势线
    pivot_type: 'low'（多头趋势线）或'high'（空头趋势线）
    """
    if len(df) < lookback:
        return None
    
    # 寻找关键点
    if pivot_type == 'low':
        # 寻找波段低点
        pivots = find_valley_pivots(df['low'], lookback)
    else:
        # 寻找波段高点
        pivots = find_peak_pivots(df['high'], lookback)
    
    if len(pivots) < min_pivots:
        return None
    
    # 线性回归拟合
    x = np.arange(len(pivots))
    y = [df.iloc[p]['low'] if pivot_type == 'low' else df.iloc[p]['high'] for p in pivots]
    
    slope, intercept = np.polyfit(x, y, 1)
    
    # 计算趋势线在当前K线的位置
    current_position = slope * (len(pivots) - 1) + intercept
    
    return {
        'slope': slope,
        'intercept': intercept,
        'current_position': current_position,
        'pivot_indices': pivots,
        'type': 'bullish_trend_line' if pivot_type == 'low' else 'bearish_trend_line'
    }

# 趋势线突破判断
def assess_trend_line_break(df, trend_line, current_idx, threshold=0.001):
    """
    评估趋势线突破
    threshold: 突破阈值（百分比）
    """
    current_bar = df.iloc[current_idx]
    line_position = trend_line['current_position']
    
    if trend_line['type'] == 'bullish_trend_line':
        # 多头趋势线突破（向下突破）
        if current_bar['low'] < line_position * (1 - threshold):
            return {
                'break_type': 'breakdown',
                'strength': (line_position - current_bar['low']) / line_position,
                'is_significant': True
            }
    else:
        # 空头趋势线突破（向上突破）
        if current_bar['high'] > line_position * (1 + threshold):
            return {
                'break_type': 'breakout',
                'strength': (current_bar['high'] - line_position) / line_position,
                'is_significant': True
            }
    
    return {'break_type': 'none', 'strength': 0, 'is_significant': False}

# 趋势线回撤入场信号
def trend_line_pullback_signal(df, trend_line, current_idx):
    """
    趋势线回撤入场信号
    """
    if trend_line['type'] != 'bullish_trend_line':
        return None  # 暂只处理多头趋势线
    
    current_bar = df.iloc[current_idx]
    line_position = trend_line['current_position']
    
    # 检查是否回撤到趋势线附近
    distance_to_line = abs(current_bar['low'] - line_position) / line_position
    
    if distance_to_line < 0.005:  # 0.5%范围内
        # 检查反转信号
        reversal_signal = identify_reversal_bar(df, current_idx, direction='bullish')
        
        if reversal_signal:
            return {
                'signal_type': 'trend_line_pullback',
                'direction': 'bullish',
                'entry_price': current_bar['high'] + 0.01,  # 信号棒上方入场
                'stop_loss': current_bar['low'] - 0.01,
                'confidence': 0.7
            }
    
    return None
```

### 第14章：趋势通道线
#### 核心概念
1. **趋势通道线定义**：与趋势线平行，位于价格行为另一侧的直线。
2. **绘制方法**：
   - 趋势线平行线
   - 独立构造（利用波段低点绘制空头趋势通道线）
3. **趋势通道线过冲**：
   - 价格短暂突破通道线，常引发反转
   - 与楔形反转相似
   - 是逆势交易的重要触发信号
4. **通道类型**：
   - 平行通道
   - 楔形（收敛通道）
   - 扩张通道
5. **机构行为解释**：通道线过冲处的反转是机构交易思想的体现。

#### 可量化规则
```python
# 趋势通道线绘制
def draw_trend_channel_line(df, trend_line, contain_all=True):
    """
    绘制趋势通道线（趋势线的平行线）
    contain_all: 是否包含趋势线两点之间的所有棒线
    """
    if trend_line['type'] == 'bullish_trend_line':
        # 多头通道：趋势通道线在上方
        # 寻找趋势线两点之间的最高点
        start_idx, end_idx = trend_line['pivot_indices'][0], trend_line['pivot_indices'][-1]
        window_df = df.iloc[start_idx:end_idx+1]
        
        if contain_all:
            reference_point = window_df['high'].max()
        else:
            # 使用波段高点
            reference_point = window_df.iloc[-1]['high']
        
        # 计算平行线
        channel_line = {
            'slope': trend_line['slope'],
            'intercept': reference_point - trend_line['slope'] * (len(window_df) - 1),
            'type': 'bullish_channel_line',
            'reference_point': reference_point
        }
    
    else:
        # 空头通道：趋势通道线在下方
        start_idx, end_idx = trend_line['pivot_indices'][0], trend_line['pivot_indices'][-1]
        window_df = df.iloc[start_idx:end_idx+1]
        
        if contain_all:
            reference_point = window_df['low'].min()
        else:
            reference_point = window_df.iloc[-1]['low']
        
        channel_line = {
            'slope': trend_line['slope'],
            'intercept': reference_point - trend_line['slope'] * (len(window_df) - 1),
            'type': 'bearish_channel_line',
            'reference_point': reference_point
        }
    
    return channel_line

# 趋势通道线过冲检测
def detect_channel_line_overshoot(df, channel_line, current_idx, overshoot_threshold=0.002):
    """
    检测趋势通道线过冲
    """
    current_bar = df.iloc[current_idx]
    line_position = channel_line['slope'] * current_idx + channel_line['intercept']
    
    if channel_line['type'] == 'bullish_channel_line':
        # 多头通道线过冲（向上突破）
        if current_bar['high'] > line_position * (1 + overshoot_threshold):
            # 检查是否反转
            next_bars = df.iloc[current_idx+1:current_idx+3] if current_idx + 3 < len(df) else None
            
            if next_bars is not None:
                # 检查后续棒线是否反转下跌
                reversal = False
                for i in range(len(next_bars)):
                    if next_bars.iloc[i]['close'] < next_bars.iloc[i]['open']:
                        reversal = True
                        break
                
                if reversal:
                    return {
                        'overshoot': True,
                        'strength': (current_bar['high'] - line_position) / line_position,
                        'reversal': True,
                        'signal_type': 'channel_line_overshoot_reversal'
                    }
    
    else:
        # 空头通道线过冲（向下突破）
        if current_bar['low'] < line_position * (1 - overshoot_threshold):
            # 检查是否反转
            next_bars = df.iloc[current_idx+1:current_idx+3] if current_idx + 3 < len(df) else None
            
            if next_bars is not None:
                # 检查后续棒线是否反转上涨
                reversal = False
                for i in range(len(next_bars)):
                    if next_bars.iloc[i]['close'] > next_bars.iloc[i]['open']:
                        reversal = True
                        break
                
                if reversal:
                    return {
                        'overshoot': True,
                        'strength': (line_position - current_bar['low']) / line_position,
                        'reversal': True,
                        'signal_type': 'channel_line_overshoot_reversal'
                    }
    
    return {'overshoot': False}

# 楔形识别（收敛通道）
def identify_wedge_pattern(df, lookback=30):
    """
    识别楔形形态（收敛通道）
    """
    # 绘制趋势线和趋势通道线
    trend_line = draw_trend_line(df, pivot_type='low', lookback=lookback)
    channel_line = draw_trend_channel_line(df, trend_line, contain_all=True)
    
    if trend_line and channel_line:
        # 检查斜率差异（楔形应具有不同的斜率）
        slope_diff = abs(trend_line['slope'] - channel_line['slope'])
        
        # 检查收敛性
        start_distance = abs(
            (trend_line['slope'] * 0 + trend_line['intercept']) - 
            (channel_line['slope'] * 0 + channel_line['intercept'])
        )
        
        end_distance = abs(
            (trend_line['slope'] * (lookback-1) + trend_line['intercept']) - 
            (channel_line['slope'] * (lookback-1) + channel_line['intercept'])
        )
        
        is_converging = end_distance < start_distance * 0.8  # 距离缩小20%以上
        
        if slope_diff > 0.0001 and is_converging:
            return {
                'pattern_type': 'wedge',
                'trend_line': trend_line,
                'channel_line': channel_line,
                'convergence_ratio': end_distance / start_distance,
                'direction': 'rising' if trend_line['slope'] > 0 else 'falling'
            }
    
    return None
```

### 第15章：通道
#### 核心概念
1. **通道定义**：价格被限制在一对平行线之间的交易区间。
2. **通道类型**：
   - 趋势通道（倾斜）
   - 交易区间（水平）
   - 三角形（收敛或扩张）
   - 楔形（收敛的趋势通道）
3. **通道中的交易策略**：
   - 紧凑通道（斜率大）：更像趋势，只做顺势交易
   - 宽幅通道（斜率小）：更像交易区间，可从两个方向交易
4. **真空效应**：通道顶部和底部的价格运动由交易者预期驱动。
5. **通道演变**：
   - 通常需要至少两条腿确认通道存在
   - 多头通道通常至少包含三次上推
   - 很多通道在第三次推后结束

#### 可量化规则

### 第21章：尖峰和通道趋势
#### 核心概念
1. **尖峰和通道趋势定义**：
   - 最常见的趋势类型，几乎存在于每天和每轮趋势中
   - 由两个阶段组成：尖峰期（快速运动）和通道期（忧虑不确定）
   - 较小的尖峰和通道趋势常嵌套在较大的趋势中

2. **尖峰期特征**：
   - 由一条或多条趋势棒构成，发出总在场内信号
   - 紧迫感强，交易者加大头寸规模
   - 本质是逃逸缺口，市场快速从一个价位运动到另一个价位
   - 通常在第一个小时内形成，常出现在当天最初几棒
   - 突破越强，通道坚持到底的可能性越大

3. **通道期特征**：
   - 忧虑和不确定感，双向交易增多
   - 像趋势型交易区间一样交易：在多头通道中，在前一棒低点下方买进；在空头通道中，在波段高点或先前棒线高点上方卖出
   - 通道极少顺势突破，即使突破也常在5棒内失败

4. **关键规则**：
   - 尖峰在第一个暂停或回撤棒处结束
   - 回撤后三种可能：趋势恢复（最常见）、市场进入交易区间、趋势反转
   - 通道常在测量运动目标处结束，常在三推结束
   - 如果通道逆势突破，不要立即入场，等待回撤（如：多头尖峰和通道向下突破后，伺机在更低的高点做空）
   - 市场通常会回调至通道起点，测试缺口（所有尖峰都是缺口）

5. **测量运动目标**：
   - 尖峰高度可作为通道的测量目标
   - 从尖峰第一棒的开盘/最低价到尖峰最后一棒的收盘/最高价
   - 腿1=腿2运动：尖峰是腿1，通道是腿2
   - 测量运动目标更多作为获利了结区，而非反转交易入场区

6. **交易策略**：
   - 尖峰期间：顺势交易，利用紧迫感
   - 通道期间：像交易区间一样交易，双向操作但以顺势为主
   - 避免在强趋势中逆势交易，除非出现清晰的二次入场信号
   - 在测量运动目标处考虑获利了结

```python
def identify_spike(df, lookback=20, min_trend_bars=3, min_momentum_ratio=0.7):

def find_spike_start(df, end_idx, direction, lookback):
    """寻找尖峰起点"""
    for i in range(end_idx, max(0, end_idx-lookback), -1):
        bar = df.iloc[i]
        prev_bar = df.iloc[i-1] if i > 0 else bar
        
        if direction == 'bullish':
            # 寻找趋势开始点：前一棒不是多头趋势棒
            is_trend_bar = bar['close'] > bar['open'] and abs(bar['close'] - bar['open']) / (bar['high'] - bar['low']) > 0.6
            is_prev_trend_bar = prev_bar['close'] > prev_bar['open'] and abs(prev_bar['close'] - prev_bar['open']) / (prev_bar['high'] - prev_bar['low']) > 0.6
            
            if is_trend_bar and not is_prev_trend_bar:
                return i
        else:
            # 空头对称逻辑
            is_trend_bar = bar['close'] < bar['open'] and abs(bar['close'] - bar['open']) / (bar['high'] - bar['low']) > 0.6
            is_prev_trend_bar = prev_bar['close'] < prev_bar['open'] and abs(prev_bar['close'] - prev_bar['open']) / (prev_bar['high'] - prev_bar['low']) > 0.6
            
            if is_trend_bar and not is_prev_trend_bar:
                return i
    
    return max(0, end_idx - lookback)

def find_pullback_end(df, spike_end, direction, max_bars):
    """寻找回撤结束点（通道起点）"""
    if direction == 'bullish':
        # 寻找回撤低点
        for i in range(spike_end + 1, min(len(df), spike_end + max_bars + 1)):
            bar = df.iloc[i]
            
            # 检查是否出现反转信号
            reversal_signal = identify_reversal_bar(df, i, direction='bullish')
            
            if reversal_signal:
                return i
            
            # 或者回撤达到一定深度
            if i > spike_end + 3:
                pullback_depth = (df.iloc[spike_end]['close'] - bar['low']) / df.iloc[spike_end]['close']
                if pullback_depth > 0.005:  # 0.5%回撤
                    return i
    else:
        # 空头对称逻辑
        for i in range(spike_end + 1, min(len(df), spike_end + max_bars + 1)):
            bar = df.iloc[i]
            
            reversal_signal = identify_reversal_bar(df, i, direction='bearish')
            
            if reversal_signal:
                return i
            
            if i > spike_end + 3:
                pullback_depth = (bar['high'] - df.iloc[spike_end]['close']) / df.iloc[spike_end]['close']
                if pullback_depth > 0.005:
                    return i
    
    return None

def calculate_overlap_ratio(df_slice):
    """计算棒线重叠比例"""
    if len(df_slice) < 2:
        return 0
    
    overlap_count = 0
    
    for i in range(1, len(df_slice)):
        prev_bar = df_slice.iloc[i-1]
        curr_bar = df_slice.iloc[i]
        
        # 检查实体重叠
        prev_high = max(prev_bar['open'], prev_bar['close'])
        prev_low = min(prev_bar['open'], prev_bar['close'])
        curr_high = max(curr_bar['open'], curr_bar['close'])
        curr_low = min(curr_bar['open'], curr_bar['close'])
        
        if curr_low <= prev_high and curr_high >= prev_low:
            overlap_count += 1
    
    return overlap_count / (len(df_slice) - 1)
```

### 第22章：趋势型交易区间日
#### 核心概念
1. **趋势型交易区间日定义**：
   - 开盘区间约为最近几日平均区间的1/3到1/2
   - 一两个小时后出现突破，然后市场再次形成交易区间
   - 相比尖峰和通道趋势日更为常见（约两倍频率）
   - 每天至少有一个尖峰和通道波段，但整体日结构更多表现为趋势型交易区间

2. **关键特征**：
   - 开盘区间持续1-3小时，波幅约为平均日区间的一半
   - 紧迫感缺乏，预示不大可能成为强趋势日
   - 初始交易区间具有"磁力"，可能将突破拉回形成另一个交易区间
   - 尖峰通常只有1-3棒，棒线较短，影线较长，彼此多重叠
   - 回撤横盘整理5-10棒，或形成弱势通道

3. **与尖峰和通道趋势日的区别**：
   - **尖峰和通道趋势日**：尖峰大而快，棒线短影线、少重叠，紧迫感强，应专注于顺势交易和波段交易
   - **趋势型交易区间日**：尖峰小而慢，棒线长影线、多重叠，忧虑感强，可在两个方向交易，更多刮头皮机会
   - 区分意义：使用不同的交易方法

4. **交易策略**：
   - 可在两个方向上交易，寻找更多刮头皮机会
   - 突破处入场胜率较低，更好在突破回撤或区间另一侧的较早反转处入场
   - 如果确信将演变为趋势型交易区间日且突破力度强，可考虑在突破棒形成过程中入场
   - 到达测量运动目标区后，市场可能转变为第二个交易区间，此时应从趋势型交易转为交易区间型交易
   - 不应把跟踪止损调至波段低点下方（容易被击中）

5. **测量运动目标**：
   - 初始交易区间高度可作为测量运动基础
   - 突破后目标约为两倍于日区间的运动
   - 市场到达测量运动目标区后常转变为第二个交易区间
   - 每个反转背后都有基于某种测量的电脑算法

6. **迟到的突破**：
   - 有时初始交易区间约为平均日区间一半，但突破到最后一小时才出现
   - 全天都是突破进入趋势型交易区间日的架构
   - 即使时间不足，也可能形成迟到的突破

```python
def identify_recent_trading_range(df, start_idx, end_idx):
    """识别最近的交易区间"""
    window = df.iloc[start_idx:end_idx+1]
    
    if len(window) < 10:
        return None
    
    # 寻找价格聚集区域
    price_std = window['close'].std()
    price_mean = window['close'].mean()
    
    # 如果价格波动较小，可能是交易区间
    if price_std / price_mean < 0.01:  # 波动小于1%
        return {
            'high': window['high'].max(),
            'low': window['low'].min(),
            'mean': price_mean,
            'std': price_std,
            'bars_count': len(window)
        }
    
    return None

def identify_recent_breakout(df, current_idx, lookback=30):
    """识别最近的突破"""
    if current_idx < lookback:
        return None
    
    window = df.iloc[current_idx-lookback:current_idx+1]
    
    # 寻找价格突破最近区间
    first_half = window.iloc[:lookback//2]
    second_half = window.iloc[lookback//2:]
    
    first_high = first_half['high'].max()
    first_low = first_half['low'].min()
    
    # 检查突破

            'breakout_price': second_half['high'].max(),
            'target_price': first_high + (first_high - first_low)  # 区间高度投射
        }
    elif second_half['low'].min() < first_low * 0.99:  # 向下突破1%

            'direction': 'bearish',
            'breakout_level': first_low,
            'breakout_price': second_half['low'].min(),
```

### 第16章：微型通道
#### 核心概念
1. **微型通道定义**：
   - 2-10条棒线组成的紧凑通道
   - 大多数棒线触及或非常接近趋势线
   - 棒线通常较小，无回撤或只有小幅回撤
2. **微型通道特点**：
   - 趋势非常强劲，第一个回撤很少导致趋势反转
   - 可能持续10棒或更多，有时被小幅回撤分隔
   - 程序交易的结果，但现代市场中所有交易都是程序交易
3. **微型通道类型**：
   - 多头微型通道（向上倾斜）
   - 空头微型通道（向下倾斜）
4. **交易策略**：
   - 顺势交易为主
   - 突破常失败，失败的突破可作为逆势交易信号
   - 微型通道在趋势中作为旗形出现

#### 可量化规则

```python
def identify_micro_channel(df, lookback=10):

            # 空头微型通道向上突破（对称逻辑）

# 通道紧凑度计算
def calculate_channel_compactness(df, trend_line):
    """
    计算通道紧凑度（0-1，越高越紧凑）
    """
    if not trend_line:
        return 0
    
    # 计算棒线与趋势线的平均距离
    distances = []
    for i in range(len(df)):
        line_pos = trend_line['slope'] * i + trend_line['intercept']
        
        if trend_line['type'] == 'bullish_trend_line':
            # 多头趋势线：计算低点到趋势线的距离
            distance = abs(df.iloc[i]['low'] - line_pos) / line_pos
        else:
            # 空头趋势线：计算高点到趋势线的距离
            distance = abs(df.iloc[i]['high'] - line_pos) / line_pos
        
        distances.append(distance)
    
    avg_distance = np.mean(distances)
    
    # 转换距离为紧凑度（距离越小，紧凑度越高）
    compactness = 1.0 / (1.0 + avg_distance * 100)
    
    return compactness
```

### 第17章：水平线：波段点和其他关键价位
#### 核心概念
1. **水平线的作用**：
   - 交易区间日：提供支撑/阻力，失败的突破常引发反转
   - 趋势日：提供回撤入场点
2. **二次信号原则**：
   - 在交易区间日中，二次更高高点或更低低点是更可靠的信号
   - 第二次尝试令市场反转更可能成功
3. **强趋势中的逆势交易**：
   - 仅在趋势线突破和强反转棒出现时才考虑
   - 应作为刮头皮交易，不应分散顺势交易的注意力

```python
    sensitivity: 灵敏度参数（0.5%）

        'supports': [],
        'resistances': []
    }

        found = False
        for resistance in levels['resistances']:
            if abs(price - resistance['price']) / resistance['price'] < sensitivity:
                resistance['count'] += 1
                resistance['indices'].append(high_idx)
                found = True

        if not found:

                support['count'] += 1
                support['indices'].append(low_idx)
                found = True
                break

            levels['supports'].append({
                'price': price,
                'count': 1,
                'indices': [low_idx],
                'type': 'swing_low'
            })

    levels['resistances'] = sorted(levels['resistances'], key=lambda x: x['count'], reverse=True)
    
    return levels
```

### 第18章：趋势交易范例
#### 核心概念
1. **趋势交易入场方法**（书中列出了30多种）：
   - 使用止损单入场：需要更熟练的图表阅读能力
   - 使用限价单或市价单入场：经验丰富交易者的选择
2. **关键入场类型**：
   - 回撤入场（高点2、低点2、楔形旗形回撤）
   - 突破回撤入场
   - 趋势线/波段点入场
   - 固定跳动数回撤入场
   - 逐步加仓策略
3. **重要原则**：
   - 趋势中只应做多或持平（多头趋势），或做空或持平（空头趋势）
   - 大多数交易者不应尝试逆势交易
   - 架构的目的是最小化风险，在强趋势中可市价入场

   - 图表从一角延伸到斜对角，中间只有小幅回撤
   - 当天开盘出现大的缺口
   - 出现趋势变化的高点和低点（波段）
   - 大部分棒线都是趋势方向上的趋势棒
   - 连续棒线的实体部分只有非常小的重叠
2. **趋势强弱判断**：
   - 趋势越强，越应专注于顺势交易
   - 在强趋势中，最好的反转棒和趋势棒通常是逆势的
   - 强趋势中不需要完美的入场架构，可市价入场
3. **趋势演变**：
   - 从尖峰期到通道期到交易区间
   - 趋势变弱的征兆：更多双向交易出现

```python
# 趋势强度评估系统
class TrendStrengthAnalyzer:
    """
    趋势强度分析器 - 基于第19章的特征
    """
    
    def __init__(self):
        self.features = {
            'gap_open': {'weight': 0.1, 'threshold': 0.01},  # 1%以上缺口
            'trend_bars_ratio': {'weight': 0.15, 'threshold': 0.7},  # 70%趋势棒
            'overlap_degree': {'weight': 0.1, 'threshold': 0.3},  # 重叠度小于30%
            'no_tails': {'weight': 0.05, 'threshold': 0.5},  # 50%无尾线
            'gap_between_bars': {'weight': 0.05, 'threshold': 0.3},  # 30%棒线间有缺口
            'no_big_climax': {'weight': 0.05, 'threshold': True},
            'weak_counter_signals': {'weight': 0.1, 'threshold': 0.7},  # 70%逆势信号弱
            'small_pullbacks': {'weight': 0.1, 'threshold': 0.02},  # 回撤小于2%
            'urgency_feeling': {'weight': 0.05, 'threshold': 0.7},  # 紧迫感评分
            'consecutive_gap_bars': {'weight': 0.1, 'threshold': 10},  # 连续10棒缺口棒
            'profitable_counter_trades': {'weight': 0.05, 'threshold': 0.2},  # 可获利逆势交易少于20%
            'failed_reversals': {'weight': 0.1, 'threshold': 0.8}  # 80%反转尝试失败
        }
    
    def analyze_trend_strength(self, df, trend_direction):
        """
        综合分析趋势强度
        返回：强度评分（0-1）和特征报告
        """
        strength_score = 0
        feature_report = {}
        
        # 1. 缺口开盘
        if len(df) > 1:
            gap = abs(df.iloc[0]['open'] - df.iloc[-1]['close']) / df.iloc[-1]['close']
            gap_score = 1.0 if gap > self.features['gap_open']['threshold'] else gap / self.features['gap_open']['threshold']
            strength_score += gap_score * self.features['gap_open']['weight']
            feature_report['gap_open'] = {'score': gap_score, 'value': gap}
        
        # 2. 趋势棒比例
        trend_bars = self.calculate_trend_bars_ratio(df, trend_direction)
        trend_bars_score = min(1.0, trend_bars / self.features['trend_bars_ratio']['threshold'])
        strength_score += trend_bars_score * self.features['trend_bars_ratio']['weight']
        feature_report['trend_bars_ratio'] = {'score': trend_bars_score, 'value': trend_bars}
        
        # 3. 棒线重叠度
        overlap = self.calculate_overlap_degree(df, trend_direction)
        overlap_score = 1.0 - min(1.0, overlap / self.features['overlap_degree']['threshold'])
        strength_score += overlap_score * self.features['overlap_degree']['weight']
        feature_report['overlap_degree'] = {'score': overlap_score, 'value': overlap}
        
        # 4. 无尾线棒线比例
        no_tails_ratio = self.calculate_no_tails_ratio(df, trend_direction)
        no_tails_score = min(1.0, no_tails_ratio / self.features['no_tails']['threshold'])
        strength_score += no_tails_score * self.features['no_tails']['weight']
        feature_report['no_tails_ratio'] = {'score': no_tails_score, 'value': no_tails_ratio}
        
        # 5. 棒线间缺口比例
        gap_ratio = self.calculate_gap_between_bars(df, trend_direction)
        gap_score = min(1.0, gap_ratio / self.features['gap_between_bars']['threshold'])
        strength_score += gap_score * self.features['gap_between_bars']['weight']
        feature_report['gap_between_bars'] = {'score': gap_score, 'value': gap_ratio}
        
        # 6. 无大型高潮
        big_climax = self.has_big_climax(df, trend_direction)
        no_climax_score = 1.0 if not big_climax else 0.0
        strength_score += no_climax_score * self.features['no_big_climax']['weight']
        feature_report['no_big_climax'] = {'score': no_climax_score, 'value': not big_climax}
        
        # 7. 逆势信号强弱
        weak_counter = self.analyze_counter_signals(df, trend_direction)
        weak_counter_score = min(1.0, weak_counter / self.features['weak_counter_signals']['threshold'])
        strength_score += weak_counter_score * self.features['weak_counter_signals']['weight']
        feature_report['weak_counter_signals'] = {'score': weak_counter_score, 'value': weak_counter}
        
        # 8. 回撤幅度
        pullback_size = self.analyze_pullback_size(df, trend_direction)
        small_pullbacks_score = 1.0 - min(1.0, pullback_size / self.features['small_pullbacks']['threshold'])
        strength_score += small_pullbacks_score * self.features['small_pullbacks']['weight']
        feature_report['small_pullbacks'] = {'score': small_pullbacks_score, 'value': pullback_size}
        
        # 9. 紧迫感评分（基于价格加速度）
        urgency = self.calculate_urgency_feeling(df, trend_direction)
        urgency_score = min(1.0, urgency / self.features['urgency_feeling']['threshold'])
        strength_score += urgency_score * self.features['urgency_feeling']['weight']
        feature_report['urgency_feeling'] = {'score': urgency_score, 'value': urgency}
        
        # 10. 连续缺口棒
        consecutive_gap = self.count_consecutive_gap_bars(df, trend_direction)
        gap_bars_score = min(1.0, consecutive_gap / self.features['consecutive_gap_bars']['threshold'])
        strength_score += gap_bars_score * self.features['consecutive_gap_bars']['weight']
        feature_report['consecutive_gap_bars'] = {'score': gap_bars_score, 'value': consecutive_gap}
        
        # 11. 可获利逆势交易比例
        profitable_counter = self.estimate_profitable_counter_trades(df, trend_direction)
        profitable_counter_score = 1.0 - min(1.0, profitable_counter / self.features['profitable_counter_trades']['threshold'])
        strength_score += profitable_counter_score * self.features['profitable_counter_trades']['weight']
        feature_report['profitable_counter_trades'] = {'score': profitable_counter_score, 'value': profitable_counter}
        
        # 12. 失败的反转尝试
        failed_reversals = self.analyze_failed_reversals(df, trend_direction)
        failed_reversals_score = min(1.0, failed_reversals / self.features['failed_reversals']['threshold'])
        strength_score += failed_reversals_score * self.features['failed_reversals']['weight']
        feature_report['failed_reversals'] = {'score': failed_reversals_score, 'value': failed_reversals}
        
        return {
            'strength_score': min(1.0, strength_score),
            'trend_direction': trend_direction,
            'feature_report': feature_report,
            'strength_level': self.classify_strength_level(min(1.0, strength_score))
        }
    
    def classify_strength_level(self, score):
        """
        分类趋势强度级别
        """
        if score >= 0.8:
            return 'very_strong'
        elif score >= 0.6:
            return 'strong'
        elif score >= 0.4:
            return 'moderate'
        elif score >= 0.2:
            return 'weak'
        else:
            return 'very_weak'
    
    def calculate_trend_bars_ratio(self, df, trend_direction):
        """
        计算趋势棒比例
        """
        if trend_direction == 'bullish':
            trend_bars = (df['close'] > df['open']).sum()
        else:
            trend_bars = (df['close'] < df['open']).sum()
        
        return trend_bars / len(df)
    
    def calculate_overlap_degree(self, df, trend_direction):
        """
        计算棒线重叠度
        """
        overlap_count = 0
        
        for i in range(1, len(df)):
            prev_bar = df.iloc[i-1]
            curr_bar = df.iloc[i]
            
            # 检查实体重叠
            if trend_direction == 'bullish':
                if curr_bar['low'] < prev_bar['high']:
                    overlap_count += 1
            else:
                if curr_bar['high'] > prev_bar['low']:
                    overlap_count += 1
        
        return overlap_count / (len(df) - 1)
    
    def calculate_no_tails_ratio(self, df, trend_direction):
        """
        计算无尾线棒线比例
        """
        no_tails_count = 0
        
        for i in range(len(df)):
            bar = df.iloc[i]
            bar_range = bar['high'] - bar['low']
            
            if trend_direction == 'bullish':
                # 多头趋势：检查下尾线
                lower_tail = bar['open'] - bar['low']
                if lower_tail / bar_range < 0.1:  # 下尾线小于10%
                    no_tails_count += 1
            else:
                # 空头趋势：检查上尾线
                upper_tail = bar['high'] - bar['open']
                if upper_tail / bar_range < 0.1:  # 上尾线小于10%
                    no_tails_count += 1
        
        return no_tails_count / len(df)
    
    def calculate_gap_between_bars(self, df, trend_direction):
        """
        计算棒线间缺口比例
        """
        gap_count = 0
        
        for i in range(1, len(df)):
            prev_bar = df.iloc[i-1]
            curr_bar = df.iloc[i]
            
            if trend_direction == 'bullish':
                if curr_bar['low'] > prev_bar['high']:
                    gap_count += 1
            else:
                if curr_bar['high'] < prev_bar['low']:
                    gap_count += 1
        
        return gap_count / (len(df) - 1)
    
    def has_big_climax(self, df, trend_direction):
        """
        检查是否有大型高潮
        """
        # 寻找异常大的趋势棒
        avg_body_size = abs(df['close'] - df['open']).mean()
        std_body_size = abs(df['close'] - df['open']).std()
        
        big_bars = abs(df['close'] - df['open']) > avg_body_size + 2 * std_body_size
        
        return big_bars.any()
    
    def analyze_counter_signals(self, df, trend_direction):
        """
        分析逆势信号强弱
        """
        counter_strength = 0
        counter_count = 0
        
        for i in range(1, len(df)):
            if trend_direction == 'bullish':
                # 多头趋势中的空头信号
                if df.iloc[i]['close'] < df.iloc[i]['open']:
                    counter_count += 1
                    # 计算空头信号强度
                    body_size = abs(df.iloc[i]['close'] - df.iloc[i]['open'])
                    bar_range = df.iloc[i]['high'] - df.iloc[i]['low']
                    strength = body_size / bar_range
                    counter_strength += strength
            else:
                # 空头趋势中的多头信号
                if df.iloc[i]['close'] > df.iloc[i]['open']:
                    counter_count += 1
                    body_size = abs(df.iloc[i]['close'] - df.iloc[i]['open'])
                    bar_range = df.iloc[i]['high'] - df.iloc[i]['low']
                    strength = body_size / bar_range
                    counter_strength += strength
        
        if counter_count == 0:
            return 1.0  # 无逆势信号，强度最高
        
        return 1.0 - (counter_strength / counter_count)  # 逆势信号越弱，评分越高
    
    def analyze_pullback_size(self, df, trend_direction):
        """
        分析回撤幅度
        """
        if trend_direction == 'bullish':
            # 计算从高点的最大回撤
            highs = df['high'].rolling(5).max()
            current_lows = df['low']
            pullbacks = (highs - current_lows) / highs
        else:
            # 计算从低点的最大反弹
            lows = df['low'].rolling(5).min()
            current_highs = df['high']
            pullbacks = (current_highs - lows) / lows
        
        return pullbacks.mean()
    
    def calculate_urgency_feeling(self, df, trend_direction):
        """
        计算紧迫感评分（基于价格加速度）
        """
        if len(df) < 3:
            return 0.5
        
        if trend_direction == 'bullish':
            prices = df['close']
        else:
            prices = df['close'] * -1  # 空头趋势中价格下跌视为"上涨"
        
        # 计算价格加速度
        returns = prices.pct_change().dropna()
        if len(returns) < 2:
            return 0.5
        
        acceleration = returns.diff().mean()
        
        # 将加速度转换为0-1评分
        urgency_score = 1.0 / (1.0 + np.exp(-10 * acceleration))
        
        return urgency_score
    
    def count_consecutive_gap_bars(self, df, trend_direction):
        """
        计算连续缺口棒数量
        """
        max_consecutive = 0
        current_consecutive = 0
        
        for i in range(1, len(df)):
            prev_close = df.iloc[i-1]['close']
            curr_open = df.iloc[i]['open']
            
            if trend_direction == 'bullish':
                if curr_open > prev_close:
                    current_consecutive += 1
                else:
                    max_consecutive = max(max_consecutive, current_consecutive)
                    current_consecutive = 0
            else:
                if curr_open < prev_close:
                    current_consecutive += 1
                else:
                    max_consecutive = max(max_consecutive, current_consecutive)
                    current_consecutive = 0
        
        return max(max_consecutive, current_consecutive)
    
    def estimate_profitable_counter_trades(self, df, trend_direction):
        """
        估计可获利逆势交易比例
        """
        # 简化估计：基于逆势信号的持续性和幅度
        profitable_estimate = 0
        
        for i in range(1, len(df)):
            if trend_direction == 'bullish':
                # 检查空头信号
                if df.iloc[i]['close'] < df.iloc[i]['open']:
                    # 检查后续棒线
                    if i + 1 < len(df):
                        if df.iloc[i+1]['close'] > df.iloc[i+1]['open']:
                            profitable_estimate += 1  # 逆势交易可能获利
            else:
                # 检查多头信号
                if df.iloc[i]['close'] > df.iloc[i]['open']:
                    if i + 1 < len(df):
                        if df.iloc[i+1]['close'] < df.iloc[i+1]['open']:
                            profitable_estimate += 1
        
        total_counter_signals = (df['close'] < df['open']).sum() if trend_direction == 'bullish' else (df['close'] > df['open']).sum()
        
        if total_counter_signals == 0:
            return 0.0
        
        return profitable_estimate / total_counter_signals
    
    def analyze_failed_reversals(self, df, trend_direction):
        """
        分析失败的反转尝试
        """
        failed_reversals = 0
        total_reversal_attempts = 0
        
        for i in range(2, len(df)):
            if trend_direction == 'bullish':
                # 检查空头反转尝试
                if df.iloc[i]['close'] < df.iloc[i]['open']:  # 空头棒
                    # 检查是否为反转尝试
                    if df.iloc[i-1]['close'] > df.iloc[i-1]['open']:  # 前一棒为多头
                        total_reversal_attempts += 1
                        # 检查是否失败（后续恢复上涨）
                        if i + 1 < len(df) and df.iloc[i+1]['close'] > df.iloc[i+1]['open']:
                            failed_reversals += 1
            else:
                # 检查多头反转尝试
                if df.iloc[i]['close'] > df.iloc[i]['open']:  # 多头棒
                    if df.iloc[i-1]['close'] < df.iloc[i-1]['open']:  # 前一棒为空头
                        total_reversal_attempts += 1
                        if i + 1 < len(df) and df.iloc[i+1]['close'] < df.iloc[i+1]['open']:
                            failed_reversals += 1
        
        if total_reversal_attempts == 0:
            return 1.0  # 无反转尝试，视为全部"失败"
        
        return failed_reversals / total_reversal_attempts
```

### 第20章：两条腿形态
#### 核心概念
1. **两条腿原则**：
   - 市场经常两次尝试做某件事情
   - 所有运动一般都可再分为两波较小运动
   - 两次尝试失败后，市场通常会做反向尝试
2. **腿的结构**：
   - 每条大腿由较小的腿构成
   - 与三推形态有重叠，但两条腿观点对交易者更有意义
   - 趋势线突破是一条新腿的起点
3. **交易意义**：
   - 提供健全的交易逻辑和充足的交易机会
   - 帮助理解市场结构和反转点

#### 可量化规则
```python
# 两条腿形态识别
class TwoLegPatternAnalyzer:
    """
    两条腿形态分析器
    """
    
    def __init__(self):
        self.min_leg_ratio = 0.3  # 最小腿长比例
        self.max_leg_ratio = 3.0  # 最大腿长比例
    
    def identify_two_leg_pattern(self, df, current_idx, pattern_type='pullback'):
        """
        识别两条腿形态
        pattern_type: 'pullback'（回撤）, 'trend'（趋势）, 'reversal'（反转）
        """
        if current_idx < 20:
            return None
        
        if pattern_type == 'pullback':
            return self.identify_two_leg_pullback(df, current_idx)
        elif pattern_type == 'trend':
            return self.identify_two_leg_trend(df, current_idx)
        elif pattern_type == 'reversal':
            return self.identify_two_leg_reversal(df, current_idx)
        
        return None
    
    def identify_two_leg_pullback(self, df, current_idx):
        """
        识别两条腿回撤
        """
        # 寻找趋势方向
        trend_direction = 'bullish' if df['close'].iloc[current_idx] > df['close'].iloc[current_idx-10] else 'bearish'
        
        if trend_direction == 'bullish':
            # 多头趋势中的两条腿回撤
            # 寻找第一个高点
            high_1_idx = self.find_swing_high(df, current_idx, lookback=20)
            if high_1_idx is None:
                return None
            
            # 寻找第一个低点（腿1终点）
            low_1_idx = self.find_swing_low_between(df, high_1_idx, current_idx)
            if low_1_idx is None:
                return None
            
            # 寻找第二个高点（腿2起点）
            high_2_idx = self.find_swing_high_between(df, low_1_idx, current_idx)
            if high_2_idx is None:
                return None
            
            # 寻找当前低点（腿2可能终点）
            low_2_idx = current_idx
            
            # 检查腿2是否低于腿1
            if df.iloc[low_2_idx]['low'] >= df.iloc[low_1_idx]['low']:
                return None  # 不是更低低点
            
            # 计算腿的长度比例
            leg_1_length = df.iloc[high_1_idx]['high'] - df.iloc[low_1_idx]['low']
            leg_2_length = df.iloc[high_2_idx]['high'] - df.iloc[low_2_idx]['low']
            
            length_ratio = leg_2_length / leg_1_length
            
            if self.min_leg_ratio <= length_ratio <= self.max_leg_ratio:
                return {
                    'pattern_type': 'two_leg_pullback',
                    'trend_direction': 'bullish',
                    'leg_1': {'high_idx': high_1_idx, 'low_idx': low_1_idx, 'length': leg_1_length},
                    'leg_2': {'high_idx': high_2_idx, 'low_idx': low_2_idx, 'length': leg_2_length},
                    'length_ratio': length_ratio,
                    'is_complete': self.is_pullback_complete(df, low_2_idx, trend_direction)
                }
        
        else:
            # 空头趋势中的两条腿反弹（对称逻辑）
            pass
        
        return None
    
    def identify_two_leg_trend(self, df, current_idx):
        """
        识别两条腿趋势
        """
        # 检查是否形成趋势
        if current_idx < 30:
            return None
        
        # 寻找趋势起点
        start_idx = self.find_trend_start(df, current_idx)
        if start_idx is None:
            return None
        
        # 寻找第一个波段高点/低点（腿1终点）
        if df['close'].iloc[current_idx] > df['close'].iloc[start_idx]:  # 多头趋势
            leg_1_high_idx = self.find_swing_high_between(df, start_idx, current_idx)
            if leg_1_high_idx is None:
                return None
            
            # 寻找回撤低点
            pullback_low_idx = self.find_swing_low_between(df, leg_1_high_idx, current_idx)
            if pullback_low_idx is None:
                return None
            
            # 检查腿2是否形成
            if df['close'].iloc[current_idx] > df.iloc[leg_1_high_idx]['high']:
                # 腿2超越腿1高点
                return {
                    'pattern_type': 'two_leg_trend',
                    'trend_direction': 'bullish',
                    'start_idx': start_idx,
                    'leg_1_high_idx': leg_1_high_idx,
                    'pullback_low_idx': pullback_low_idx,
                    'current_high_idx': current_idx,
                    'is_extended': True
                }
            else:
                # 腿2未超越腿1高点
                return {
                    'pattern_type': 'two_leg_trend',
                    'trend_direction': 'bullish',
                    'start_idx': start_idx,
                    'leg_1_high_idx': leg_1_high_idx,
                    'pullback_low_idx': pullback_low_idx,
                    'current_high_idx': current_idx,
                    'is_extended': False
                }
        
        return None
    
    def identify_two_leg_reversal(self, df, current_idx):
        """
        识别两条腿反转
        """
        # 检查是否形成双重顶/底
        double_pattern = self.identify_double_pattern(df, current_idx)
        
        if double_pattern:
            # 检查是否为两次尝试失败
            if double_pattern['pattern_type'] in ['double_top', 'double_bottom']:
                # 计算两次尝试的时间间隔
                time_gap = double_pattern['second_peak_idx'] - double_pattern['first_peak_idx']
                
                if 5 <= time_gap <= 30:  # 合理的间隔
                    return {
                        'pattern_type': 'two_leg_reversal',
                        'double_pattern': double_pattern,
                        'is_failed_attempt': self.analyze_attempt_strength(df, double_pattern)
                    }
        
        return None
    
    def find_swing_high(self, df, end_idx, lookback=20):
        """
        寻找波段高点
        """
        start_idx = max(0, end_idx - lookback)
        
        if start_idx >= end_idx:
            return None
        
        window = df.iloc[start_idx:end_idx+1]
        high_idx = window['high'].idxmax()
        
        # 验证是否为有效高点
        if high_idx > start_idx and high_idx < end_idx:
            # 检查前后是否有更低高点
            if df.iloc[high_idx]['high'] > df.iloc[high_idx-1]['high'] and \
               df.iloc[high_idx]['high'] > df.iloc[high_idx+1]['high']:
                return high_idx
        
        return None
    
    def find_swing_low(self, df, end_idx, lookback=20, start_idx=None):
        """
        寻找波段低点
        """
        if start_idx is None:
            start_idx = max(0, end_idx - lookback)
        
        if start_idx >= end_idx:
            return None
        
        window = df.iloc[start_idx:end_idx+1]
        low_idx = window['low'].idxmin()
        
        # 验证是否为有效低点
        if low_idx > start_idx and low_idx < end_idx:
            if df.iloc[low_idx]['low'] < df.iloc[low_idx-1]['low'] and \
               df.iloc[low_idx]['low'] < df.iloc[low_idx+1]['low']:
                return low_idx
        
        return None
    
    def find_swing_high_between(self, df, start_idx, end_idx):
        """
        在两个索引间寻找波段高点
        """
        return self.find_swing_high(df, end_idx, lookback=end_idx-start_idx)
    
    def find_swing_low_between(self, df, start_idx, end_idx):
        """
        在两个索引间寻找波段低点
        """
        return self.find_swing_low(df, end_idx, lookback=end_idx-start_idx, start_idx=start_idx)
    
    def find_trend_start(self, df, current_idx):
        """
        寻找趋势起点
        """
        # 寻找最近的重要反转点
        for i in range(current_idx-1, max(0, current_idx-50), -1):
            # 检查是否出现趋势变化
            if i > 10:
                prev_trend = 'bullish' if df['close'].iloc[i] > df['close'].iloc[i-10] else 'bearish'
                curr_trend = 'bullish' if df['close'].iloc[current_idx] > df['close'].iloc[i] else 'bearish'
                
                if prev_trend != curr_trend:
                    # 检查是否有明确的趋势线突破
                    if self.has_trend_line_break(df, i, current_idx):
                        return i
        
        return None
    
    def has_trend_line_break(self, df, start_idx, end_idx):
        """
        检查是否有趋势线突破
        """
        # 简化检查：价格变化超过一定阈值
        price_change = abs(df['close'].iloc[end_idx] - df['close'].iloc[start_idx]) / df['close'].iloc[start_idx]
        
        return price_change > 0.02  # 2%以上变化
    
    def identify_double_pattern(self, df, current_idx):
        """
        识别双重顶/底形态
        """
        if current_idx < 20:
            return None
        
        # 寻找最近的波段高点
        recent_high_idx = self.find_swing_high(df, current_idx, lookback=20)
        if recent_high_idx is None:
            return None
        
        # 寻找前一个波段高点
        prev_high_idx = self.find_swing_high(df, recent_high_idx-1, lookback=20)
        if prev_high_idx is None:
            return None
        
        # 检查是否形成双重顶
        high_1 = df.iloc[prev_high_idx]['high']
        high_2 = df.iloc[recent_high_idx]['high']
        
        if abs(high_2 - high_1) / high_1 < 0.01:  # 1%以内
            # 寻找中间的低点
            low_idx = self.find_swing_low_between(df, prev_high_idx, recent_high_idx)
            
            if low_idx:
                return {
                    'pattern_type': 'double_top',
                    'first_peak_idx': prev_high_idx,
                    'second_peak_idx': recent_high_idx,
                    'valley_idx': low_idx,
                    'price_level': (high_1 + high_2) / 2
                }
        
        # 检查双重底（对称逻辑）
        recent_low_idx = self.find_swing_low(df, current_idx, lookback=20)
        if recent_low_idx:
            prev_low_idx = self.find_swing_low(df, recent_low_idx-1, lookback=20)
            
            if prev_low_idx:
                low_1 = df.iloc[prev_low_idx]['low']
                low_2 = df.iloc[recent_low_idx]['low']
                
                if abs(low_2 - low_1) / low_1 < 0.01:
                    high_idx = self.find_swing_high_between(df, prev_low_idx, recent_low_idx)
                    
                    if high_idx:
                        return {
                            'pattern_type': 'double_bottom',
                            'first_trough_idx': prev_low_idx,
                            'second_trough_idx': recent_low_idx,
                            'peak_idx': high_idx,
                            'price_level': (low_1 + low_2) / 2
                        }
        
        return None
    
    def analyze_attempt_strength(self, df, double_pattern):
        """
        分析尝试强度（是否失败）
        """
        if double_pattern['pattern_type'] == 'double_top':
            # 检查第二次尝试是否较弱
            bar_1 = df.iloc[double_pattern['first_peak_idx']]
            bar_2 = df.iloc[double_pattern['second_peak_idx']]
            
            # 计算第二次尝试的动能
            momentum_1 = bar_1['high'] - bar_1['low']
            momentum_2 = bar_2['high'] - bar_2['low']
            
            return momentum_2 < momentum_1 * 0.8  # 第二次尝试动能弱于第一次
        
        elif double_pattern['pattern_type'] == 'double_bottom':
            # 对称逻辑
            bar_1 = df.iloc[double_pattern['first_trough_idx']]
            bar_2 = df.iloc[double_pattern['second_trough_idx']]
            
            momentum_1 = bar_1['high'] - bar_1['low']
            momentum_2 = bar_2['high'] - bar_2['low']
            
            return momentum_2 < momentum_1 * 0.8
        
        return False
    
    def is_pullback_complete(self, df, low_idx, trend_direction):
        """
        判断回撤是否完成
        """
        if trend_direction == 'bullish':
            # 多头趋势回撤完成：出现反转上涨信号
            if low_idx + 1 < len(df):
                next_bar = df.iloc[low_idx + 1]
                return next_bar['close'] > next_bar['open']  # 多头棒
        
        return False
```
