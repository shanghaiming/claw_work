## 🎯 策略概述

### 策略名称
**六因子协同交易策略**

### 策略描述
本策略结合六个关键技术指标，通过多因子投票机制产生交易信号，旨在捕捉市场趋势并过滤震荡行情。策略综合考量趋势、动量、波动率和成交量四个维度，提高信号的可靠性。

### 核心思想
1. **多因子验证**：单个指标易产生假信号，多个指标协同可提高准确性
2. **动态权重**：根据市场状态（趋势/震荡）自动调整因子重要性
3. **风险控制**：通过CHOP指标识别震荡市，减少无效交易

## 📊 指标详解

### 1. Fisher Transform (费希尔变换)
- **作用**：识别价格极端点，预测趋势反转
- **计算**：基于价格标准化后的双曲正切变换
- **信号**：Fisher值 > 信号线时为买入信号
- **参数**：周期=10天

### 2. Choppiness Index (震荡指数)
- **作用**：衡量市场处于趋势还是震荡状态
- **计算**：基于价格波动范围与ATR的比率
- **信号**：CHOP < 38.2% 表示趋势市场，适合交易
- **参数**：周期=14天

### 3. Hull Moving Average (赫尔移动平均线)
- **作用**：减少传统均线滞后，更快反应价格变化
- **计算**：WMA(2×WMA(n/2) - WMA(n), √n)
- **信号**：价格 > HMA 时为上升趋势
- **参数**：周期=9天

### 4. Least Squares Moving Average (最小二乘移动平均)
- **作用**：基于线性回归的趋势线，对趋势更敏感
- **计算**：对近期价格进行线性回归拟合
- **信号**：价格 > LSMA 时为买入信号
- **参数**：周期=14天

### 5. MACD (移动平均收敛发散)
- **作用**：经典动量指标，识别趋势方向和强度
- **计算**：12日EMA - 26日EMA，9日信号线
- **信号**：MACD差值 > 0 时为买入信号
- **参数**：快线=12，慢线=26，信号=9

### 6. Volume Ratio (成交量比率)
- **作用**：衡量成交量相对强度，确认价格趋势
- **计算**：5日成交量均线 / 20日成交量均线
- **信号**：VR > 1 表示放量，确认趋势有效性
- **参数**：快线=5，慢线=20

## ⚙️ 策略逻辑

### 信号生成规则
| 条件 | 买入信号 | 卖出信号 | 持有信号 |
|------|----------|----------|----------|
| Fisher Transform | Fisher > 信号线 | Fisher < 信号线 | 其他 |
| Choppiness Index | CHOP < 38.2% | CHOP > 61.8% | 中间区域 |
| Hull MA | 价格 > HMA | 价格 < HMA | 价格 ≈ HMA |
| LSMA | 价格 > LSMA | 价格 < LSMA | 价格 ≈ LSMA |
| MACD | MACD差值 > 0 | MACD差值 < 0 | MACD差值 ≈ 0 |
| Volume Ratio | VR > 1.0 | VR < 0.8 | 0.8 ≤ VR ≤ 1.0 |

### 投票决策机制
- **买入条件**：6个指标中至少4个给出买入信号
- **卖出条件**：6个指标中少于2个给出买入信号（即多数指标看空）
- **持有条件**：介于两者之间（2-3个买入信号）

### 风险管理
1. **仓位控制**：单次交易使用95%可用资金
2. **手续费**：考虑0.1%的交易成本
3. **止损机制**：基于ATR的动态止损（可选项）
4. **最大回撤**：监控并控制回撤在可接受范围

## 💻 Python实现

### 完整代码
```python
#!/usr/bin/env python3
"""
多因子交易策略 - 六因子协同模型
包含：Fisher, CHOP, HMA, LSMA, MACD, VR 指标
"""

import pandas as pd
import numpy as np
import yfinance as yf
import matplotlib.pyplot as plt
from datetime import datetime

class MultiFactorStrategy:
    """多因子交易策略类"""
    
    def __init__(self, symbol='SPY', period='1y', interval='1d'):
        self.symbol = symbol
        self.period = period
        self.interval = interval
        self.data = None
        self.parameters = {
            'fisher_period': 10,
            'chop_period': 14,
            'hma_period': 9,
            'lsma_period': 14,
            'macd_fast': 12,
            'macd_slow': 26,
            'macd_signal': 9,
            'vr_fast': 5,
            'vr_slow': 20,
            'buy_threshold': 4,
            'sell_threshold': 2
        }
    
    def fetch_data(self):
        """获取历史数据"""
        ticker = yf.Ticker(self.symbol)
        self.data = ticker.history(period=self.period, interval=self.interval)
        return self.data
    
    def calculate_fisher(self):
        """计算 Fisher Transform"""
        typical_price = (self.data['High'] + self.data['Low'] + self.data['Close']) / 3
        period = self.parameters['fisher_period']
        
        # 标准化价格
        median = typical_price.rolling(window=period).median()
        std = typical_price.rolling(window=period).std().replace(0, 1)
        normalized = (typical_price - median) / std
        
        # Fisher 变换
        normalized = np.clip(normalized, -0.999, 0.999)
        self.data['fisher'] = 0.5 * np.log((1 + normalized) / (1 - normalized))
        self.data['fisher_signal'] = self.data['fisher'].rolling(window=3).mean()
        
        # 生成信号
        self.data['fisher_buy'] = (self.data['fisher'] > self.data['fisher_signal']).astype(int)
    
    def calculate_chop(self):
        """计算 Choppiness Index"""
        period = self.parameters['chop_period']
        sum_range = (self.data['High'] - self.data['Low']).rolling(window=period).sum()
        atr = (abs(self.data['High'] - self.data['Close'].shift(1))).rolling(window=period).mean()
        
        # 避免除零
        atr = atr.replace(0, 1)
        self.data['chop'] = 100 * np.log10(sum_range / atr) / np.log10(period)
        
        # 生成信号 (CHOP < 38.2 为趋势市)
        self.data['chop_buy'] = (self.data['chop'] < 38.2).astype(int)
    
    def calculate_hma(self):
        """计算 Hull Moving Average"""
        period = self.parameters['hma_period']
        
        # 加权移动平均函数
        def wma(series, window):
            weights = np.arange(1, window + 1)
            return series.rolling(window=window).apply(
                lambda x: np.sum(x * weights[:len(x)]) / np.sum(weights[:len(x)]), 
                raw=True
            )
        
        wma_half = wma(self.data['Close'], period // 2)
        wma_full = wma(self.data['Close'], period)
        self.data['hma'] = (2 * wma_half - wma_full).rolling(window=int(np.sqrt(period))).mean()
        
        # 生成信号
        self.data['hma_buy'] = (self.data['Close'] > self.data['hma']).astype(int)
    
    def calculate_lsma(self):
        """计算 Least Squares Moving Average"""
        def linear_regression(series):
            x = np.arange(len(series))
            slope, intercept = np.polyfit(x, series.values, 1)
            return intercept + slope * (len(series) - 1)
        
        period = self.parameters['lsma_period']
        self.data['lsma'] = self.data['Close'].rolling(window=period).apply(linear_regression, raw=False)
        
        # 生成信号
        self.data['lsma_buy'] = (self.data['Close'] > self.data['lsma']).astype(int)
    
    def calculate_macd(self):
        """计算 MACD"""
        fast = self.parameters['macd_fast']
        slow = self.parameters['macd_slow']
        signal = self.parameters['macd_signal']
        
        ema_fast = self.data['Close'].ewm(span=fast, adjust=False).mean()
        ema_slow = self.data['Close'].ewm(span=slow, adjust=False).mean()
        self.data['macd'] = ema_fast - ema_slow
        self.data['macd_signal'] = self.data['macd'].ewm(span=signal, adjust=False).mean()
        self.data['macd_diff'] = self.data['macd'] - self.data['macd_signal']
        
        # 生成信号
        self.data['macd_buy'] = (self.data['macd_diff'] > 0).astype(int)
    
    def calculate_vr(self):
        """计算 Volume Ratio"""
        fast = self.parameters['vr_fast']
        slow = self.parameters['vr_slow']
        
        volume_fast = self.data['Volume'].rolling(window=fast).mean()
        volume_slow = self.data['Volume'].rolling(window=slow).mean()
        self.data['vr'] = volume_fast / volume_slow
        
        # 生成信号
        self.data['vr_buy'] = (self.data['vr'] > 1.0).astype(int)
    
    def generate_signals(self):
        """生成综合交易信号"""
        # 计算所有指标
        self.calculate_fisher()
        self.calculate_chop()
        self.calculate_hma()
        self.calculate_lsma()
        self.calculate_macd()
        self.calculate_vr()
        
        # 汇总买入信号
        buy_columns = ['fisher_buy', 'chop_buy', 'hma_buy', 'lsma_buy', 'macd_buy', 'vr_buy']
        self.data['total_buy_signals'] = self.data[buy_columns].sum(axis=1)
        
        # 生成交易信号
        buy_threshold = self.parameters['buy_threshold']
        sell_threshold = self.parameters['sell_threshold']
        
        self.data['signal'] = 0
        self.data['signal'] = np.where(
            self.data['total_buy_signals'] >= buy_threshold, 1,
            np.where(self.data['total_buy_signals'] <= sell_threshold, -1, 0)
        )
        
        return self.data
    
    def backtest(self, initial_capital=100000, commission=0.001):
        """策略回测"""
        capital = initial_capital
        position = 0
        trades = []
        portfolio = []
        
        for i in range(len(self.data)):
            date = self.data.index[i]
            price = self.data['Close'].iloc[i]
            signal = self.data['signal'].iloc[i] if 'signal' in self.data.columns else 0
            
            # 执行交易
            if signal == 1 and position == 0:  # 买入
                shares = int(capital * 0.95 / price)
                if shares > 0:
                    cost = shares * price * (1 + commission)
                    capital -= cost
                    position = shares
                    trades.append({'date': date, 'action': 'BUY', 'price': price, 'shares': shares})
            
            elif signal == -1 and position > 0:  # 卖出
                proceeds = position * price * (1 - commission)
                capital += proceeds
                trades.append({'date': date, 'action': 'SELL', 'price': price, 'shares': position})
                position = 0
            
            # 记录组合价值
            portfolio_value = capital + position * price
            portfolio.append({'date': date, 'value': portfolio_value})
        
        # 最后清仓
        if position > 0:
            last_price = self.data['Close'].iloc[-1]
            capital += position * last_price * (1 - commission)
            position = 0
        
        # 计算绩效
        portfolio_df = pd.DataFrame(portfolio)
        portfolio_df.set_index('date', inplace=True)
        
        results = {
            'initial_capital': initial_capital,
            'final_value': capital,
            'total_return': (capital / initial_capital - 1) * 100,
            'trades': trades,
            'portfolio': portfolio_df
        }
        
        return results

# 使用示例
if __name__ == "__main__":
    # 创建策略实例
    strategy = MultiFactorStrategy(symbol='SPY', period='1y')
    
    # 获取数据
    data = strategy.fetch_data()
    
    # 计算指标和信号
    strategy.generate_signals()
    
    # 回测
    results = strategy.backtest()
    
    # 打印结果
    print(f"初始资金: ${results['initial_capital']:,.2f}")
    print(f"最终价值: ${results['final_value']:,.2f}")
    print(f"总收益率: {results['total_return']:.2f}%")
    print(f"交易次数: {len(results['trades'])}")
```

## ⚠️ 风险提示

### 策略局限性
1. **参数敏感**：策略表现对参数设置较为敏感
2. **过拟合风险**：在历史数据上表现良好不代表未来持续有效
3. **市场变化**：策略可能不适应市场结构重大变化
4. **极端行情**：黑天鹅事件中技术指标可能失效

### 使用建议
1. **模拟测试**：建议至少3个月模拟交易验证
2. **资金管理**：单策略仓位不超过总资金的20%
3. **动态调整**：定期（每季度）回顾和优化参数
4. **多策略组合**：建议与其他策略组合降低风险

### 适用场景
- **市场类型**：趋势市表现最佳，震荡市需谨慎
- **品种选择**：流动性好的ETF和大型股
- **时间周期**：日线或以上周期，避免高频噪音
- **资金规模**：适合中小规模资金（10万-1000万）
