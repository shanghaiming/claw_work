## 1. Persistency with Moving Averages
- **URL**: https://www.tradingview.com/script/eJ3uq6Ja-Persistency-with-Moving-Averages/
- **核心逻辑**: 基于Bonde的"趋势持续性"概念，追踪价格相对于关键移动平均线持续维持在上方或下方的K线数量（bars）。当价格持续停留在MA上方/下方越久，趋势延续的概率越高。不是简单计算close > previous close的次数，而是跟踪相对于MA的停留时间。
- **技术指标**: 多周期移动平均线（MA）、持续性计数器
- **策略类型**: 趋势跟踪 / 趋势强度评估
- **适用市场/时间框架**: 股票市场、日线级别
- **创新点**: 将趋势持续性（persistency）概念量化为具体可操作的指标，用MA作为动态支撑/阻力来评估趋势韧性，比传统方向判断更精细

## 2. Risk Management Engine | Anonycryptous
- **URL**: https://www.tradingview.com/script/ofFha2Mp-Risk-Management-Engine-Anonycryptous/
- **核心逻辑**: 一个专注于风险管理的指标引擎，区别于大多数只关注市场结构和进场的指标。核心目的是帮助交易者训练纪律性，提供仓位保护机制。设计为免费工具，帮助因亏损或虚假导师而盲目交易的人。
- **技术指标**: 仓位管理计算、风险参数
- **策略类型**: 风险管理 / 仓位控制
- **适用市场/时间框架**: 全市场通用
- **创新点**: 将风险管理本身做成独立指标，而非依附于任何交易策略。强调纪律训练的重要性

## 3. Trendlines with Breaks [EXCAVO]
- **URL**: https://www.tradingview.com/script/J49l5VTa-Trendlines-with-Breaks-EXCAVO/
- **核心逻辑**: 自动绘制趋势线并检测突破。使用连续摆动高低点（consecutive swing highs/lows）创建趋势线。当新枢轴点与活跃趋势线（在ATR容差范围内）对齐时，线条会延伸穿过三个或更多枢轴点而非创建重复线。
- **技术指标**: 摆动高低点检测、ATR（平均真实范围）、趋势线延伸算法
- **策略类型**: 趋势线突破 / 技术分析
- **适用市场/时间框架**: 全市场、多时间框架
- **创新点**: 使用ATR容差来判断新枢轴是否属于现有趋势线，避免重复画线；支持3+枢轴点延伸趋势线

## 4. Merged EMA Crossover & Pullback Analyzer
- **URL**: https://www.tradingview.com/script/p8mAdBcy-Merged-EMA-Crossover-Pullback-Analyzer/
- **核心逻辑**: 结合趋势跟踪和回撤分析的双重系统。EMA交叉逻辑遵循Ed Seykota的指导原则：慢线周期至少是快线的3倍。在趋势方向确认后，检测价格回撤到EMA区域的机会，提供更优入场点。
- **技术指标**: Fast EMA、Slow EMA（周期比至少3:1）、回撤检测器、波动率分析
- **策略类型**: 趋势跟踪 + 回撤入场
- **适用市场/时间框架**: 加密货币(BTC/USD)、日线
- **创新点**: 将EMA交叉信号与回撤入场分析合并为一个覆盖图，遵循Ed Seykota的EMA周期比例原则

## 6. Volumetric Structure Engine [JOAT]
- **URL**: https://www.tradingview.com/script/MD1PrIxW-Volumetric-Structure-Engine-JOAT/
- **核心逻辑**: 机构级市场结构追踪器，融合摆动点分类与实时买卖量差分析。每个确认的摆动高/低点不仅按价格测量，还按产生该波段的净成交量组成来衡量——揭示结构性移动是由真正的机构买卖驱动还是低成交量探测。分类市场结构为HH/HL（看涨）或LH/LL（看跌），检测BOS（结构突破）和ChoCH（特征变化）事件。
- **技术指标**: 摆动点检测、买卖量差分析、BOS/ChoCH结构识别、累计成交量统计
- **策略类型**: 市场结构分析 / Smart Money概念
- **适用市场/时间框架**: 全市场、多时间框架
- **创新点**: 解决了"大多数市场结构工具只画线不说质量"的问题——量化每个摆动点的成交量质量，区分真正的机构参与与流动性收割

## 9. ORB Retest Entry
- **URL**: https://www.tradingview.com/script/9X71dUke-ORB-Retest-Entry/
- **核心逻辑**: 专业的日内ORB（开盘范围突破）再测试入场工具。与传统ORB指标不同，加入了"时间限制有效性窗口"——只在市场动量仍然新鲜且可操作时生成信号。自动检测开盘范围，等待突破，然后仅在价格在用户定义的时间窗口内回测原始水平时触发信号。
- **技术指标**: 开盘范围（Opening Range Breakout）、时间窗口过滤、回测检测、Buffer百分比

- **创新点**: 时间限制有效性窗口过滤掉延迟入场，通过回测确认提高信号质量；用Buffer %避免假突破

## 13. BB Volatility Breakout Engine + TP/SL
- **URL**: https://www.tradingview.com/script/KMg1FI3i-BB-Volatility-Breakout-Engine-TP-SL/
- **核心逻辑**: 基于布林带扩张的动量指标，识别市场从低波动向高波动转换的时机。核心逻辑是BB宽度变化率（delta）：增加=波动率扩张、减少=波动率收缩。信号需满足：活跃波动率扩张 + 方向对齐（斜率） + 蜡烛方向确认 + 无同阶段前信号。自动计算入场、SL（ATR或百分比）和三个TP级别。
- **技术指标**: 布林带（Bollinger Bands）、BB宽度Delta直方图、方向斜率过滤、Histogram MA过滤、ATR止损、蜡烛方向确认
- **策略类型**: 波动率突破 / 动量交易
- **适用市场/时间框架**: 趋势市场、多时间框架
- **创新点**: 不依赖经典价格水平，而是分析BB扩张动态学；完整的交易设置自动生成（Entry + SL + TP1/2/3）；双阈值信号确认

## 16. Reversal Probability & Signals
- **URL**: https://www.tradingview.com/script/Vg1GzKm7-Reversal-Probability-Signals/
- **核心逻辑**: 使用简化KNN（K-最近邻）机器学习算法预测市场转折点（Pivot Highs/Lows）。提取8个不同长度的CCI值作为特征集（10-200期），在历史枢轴点记录这些CCI状态。每根新K线上计算当前8个CCI值与历史样本的欧几里得距离，找K个最近邻，根据多数类别输出反转概率。
- **技术指标**: 8个不同周期的CCI（10-200）、KNN分类器、欧几里得距离、双阈值信号系统
- **策略类型**: 反转交易 / 机器学习
- **适用市场/时间框架**: 全市场、多时间框架
- **创新点**: 在Pine Script内实现KNN机器学习算法；动态K值调整（根据样本量自动缩放）；两步信号逻辑（Standby >80% → Trigger跌破50%）确认动量已破

## 18. Vigilant Asset Allocation (VAA)
- **URL**: https://www.tradingview.com/script/AGViqeuJ-Vigilant-Asset-Allocation-VAA/

- **创新点**: 学术级别严谨性（引用5篇论文）；广度维度——不是逐资产独立看，而是计数进攻资产中正动量的数量；月度锁定+日内监控双模式
