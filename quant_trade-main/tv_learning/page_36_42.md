## 1. Clean Trendline Breakout Detector

- **Core Logic**: Automatically detects valid trendlines by connecting swing pivots, then fires signals when price breaks the trendline with volume confirmation. Filters out false breakouts by requiring a close beyond the trendline.
- **Technical Indicators**: Swing Pivots, Linear Regression Trendlines, Volume Filter
- **Strategy Type**: 趋势跟踪 / 突破
- **Innovation**: Automated trendline construction using pivot-to-pivot geometry. Validates breakouts with both price close AND volume threshold. Prevents repainting by using confirmed pivots only.
- **借鉴**: 趋势线的自动绘制算法——用摆动点连接+线性回归验证斜率有效性，可以量化实现传统手工画线.

## 2. BTC Kijun + RSI Strategy [75m]

- **Core Logic**: Uses Ichimoku's Kijun-sen (base line) as primary trend filter combined with RSI for momentum timing. Entry on RSI pullback to Kijun level in the direction of the higher-timeframe trend.
- **Technical Indicators**: Ichimoku Kijun-sen, RSI(14), HTF Trend Filter
- **Strategy Type**: 趋势跟踪 / 动量
- **Market/Timeframe**: BTC, 75min (custom timeframe)
- **Innovation**: Custom 75-minute timeframe specifically optimized for BTC's volatility cycle. Kijun-sen as dynamic support/resistance rather than just trend direction.
- **借鉴**: 非标准时间框架的使用思路——找到特定资产最匹配的K线周期. Kijun线作为动态支撑阻力的量化方法.

## 3. Institutional Bias Engine [Confirmed]

- **Core Logic**: Aggregates multiple institutional footprint signals (delta volume, large lot detection, VWAP deviation, order flow imbalance) into a composite bias score that indicates whether smart money is net accumulating or distributing.
- **Technical Indicators**: Delta Volume, Large Lot Filter, VWAP Deviation, Order Flow Imbalance
- **Strategy Type**: 订单流 / 流动性分析
- **Innovation**: Multi-signal institutional footprint aggregation with weighted scoring. Each signal contributes independently weighted evidence. Confirmation engine prevents false positives.
- **借鉴**: 多信号机构足迹聚合的加权评分方法——delta volume + 大单过滤 + VWAP偏离 + 订单流不平衡，每个信号独立加权贡献.

## 4. Liquidity Sweep Detector [QuantAlgo]

- **Core Logic**: Identifies liquidity pools above swing highs and below swing lows, then detects when price sweeps through these levels and reverses — a classic institutional liquidity grab pattern.
- **Technical Indicators**: Swing Pivots, Liquidity Pool Detection, Reversal Confirmation
- **Strategy Type**: 流动性分析 / 反转
- **Innovation**: Quantifies liquidity pools by measuring how much volume/interest clusters near swing extremes. Sweep detection requires both a breach AND reversal, filtering out genuine breakouts.
- **借鉴**: 流动性扫荡的量化方法——1) 识别摆动高低点附近的价格聚集区 2) 检测价格穿越这些区域后反转 3) 用成交量验证扫荡的有效性.

## 5. Adaptive Reaction Engine (ARE)

- **Core Logic**: Dynamically adjusts entry parameters based on current volatility regime. In high volatility, uses wider stops and fewer signals; in low volatility, tightens stops and increases signal sensitivity.
- **Technical Indicators**: ATR-based Volatility Regime, Adaptive Entry/Exit Parameters
- **Strategy Type**: 波动率 / 自适应
- **Innovation**: State machine with volatility-dependent parameter scaling. Not just one ATR multiplier but a full parameter set that shifts based on regime classification.
- **借鉴**: 波动率状态机的设计——高/中/低三个状态，每个状态对应一套完整的参数（止损距离、入场灵敏度、仓位大小），而非简单缩放.

## 6. Professional ATR Volatility Trading System

- **Core Logic**: Comprehensive ATR-based system that uses ATR percentile ranking to classify volatility state, then applies appropriate trading rules for each state. Includes dynamic position sizing based on ATR.
- **Technical Indicators**: ATR, ATR Percentile Rank, Dynamic Position Sizing
- **Strategy Type**: 波动率 / 风险管理
- **Innovation**: ATR percentile ranking over lookback period creates a normalized 0-100 volatility score regardless of asset or timeframe. Position sizing formula: `size = risk_capital / (ATR * multiplier)`.
- **借鉴**: ATR百分位排名将不同资产/不同周期的波动率统一到0-100尺度. 动态仓位公式: `仓位 = 风险资金 / (ATR * 乘数)`.

## 7. Macro ROC Regime

- **Core Logic**: Uses long-period Rate of Change (ROC) to classify macro market regime: expansion (strong positive ROC), contraction (near-zero ROC), recession (strong negative ROC). Each regime has different optimal strategies.
- **Technical Indicators**: ROC (long period, e.g. 50-200), Regime Classification
- **Strategy Type**: 波动率分析 / 状态分类
- **Innovation**: Instead of using ROC for timing, uses it purely for regime identification. The ROC threshold levels are adaptive, based on historical percentile distribution.
- **借鉴**: 用长周期ROC做宏观状态识别——不是用来择时，而是用来决定当前应该用什么策略类型. 自适应阈值基于历史百分位分布.

## 8. Fibonacci Volatility Cloud [JOAT]

- **Core Logic**: Combines Fibonacci retracement levels with ATR-based volatility bands to create a "cloud" that shows where price is likely to find support/resistance with statistical confidence. The cloud width adjusts with volatility.
- **Technical Indicators**: Fibonacci Levels, ATR Volatility Bands
- **Strategy Type**: 支撑阻力 / 波动率
- **Innovation**: Merges two independent frameworks (Fibonacci structure + ATR volatility) into a unified probability cloud. The cloud narrows in low-vol and widens in high-vol, naturally adjusting to market conditions.
- **借鉴**: 斐波那契结构 + ATR波动带的融合——在低波动时通道收窄（更精确的支撑阻力），高波动时通道展宽（更大的价格运动空间）.

## 9. Confluence Matrix [JOAT]

- **Core Logic**: Evaluates multiple independent analysis dimensions (trend, momentum, volume, volatility, sentiment proxy) and constructs a matrix score. Only trades when enough dimensions agree — systematic confluence quantification.
- **Technical Indicators**: Multi-dimensional scoring (Trend + Momentum + Volume + Volatility + Sentiment)
- **Strategy Type**: 多因子 / 趋势跟踪
- **Innovation**: Systematic confluence scoring: each dimension scores 0-2 (bearish/neutral/bullish), summed to a total. The matrix approach ensures dimensions are independent and non-redundant.
- **借鉴**: 多因子融合评分系统——每个维度独立评分0-2，求和后量化"共振"程度. 确保各维度独立不冗余是关键设计原则.

## 10. Simple Pyramid Strategy

- **Core Logic**: Progressive position building (pyramiding) where each additional entry requires increasingly strict confirmation. Initial entry uses basic conditions; subsequent entries need additional momentum confirmation.
- **Technical Indicators**: MA Crossover, RSI Momentum Confirmation, Progressive Entry Rules
- **Strategy Type**: 趋势跟踪 / 仓位管理
- **Innovation**: Not just "add on dips" but a structured pyramid where each layer has progressively stricter entry requirements. Position size decreases with each layer to manage risk.
- **借鉴**: 结构化加仓策略——初始仓位用基本条件，后续加仓需要额外动量确认. 每层仓位递减来管理风险.

## 11. Seykota Trend System

- **Core Logic**: Implements Ed Seykota's classic trend-following principles: long-term channel breakout entries, pyramiding, and trailing stops. Emphasizes risk management and letting profits run.
- **Technical Indicators**: Donchian Channel, Trailing Stop (ATR-based), Pyramiding Rules
- **Strategy Type**: 趋势跟踪 (经典)
- **Innovation**: Faithful implementation of Seykota's systematic approach with modern improvements: adaptive channel width based on volatility, and a maximum pyramid level to prevent over-concentration.
- **借鉴**: Seykota趋势系统的现代化实现——Donchian通道突破入场、ATR追踪止损、结构化加仓、最大持仓层数限制防止过度集中.
